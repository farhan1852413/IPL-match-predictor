# IPL Winner Predictor 🏏

A machine learning based IPL match winner prediction web application.

## 🚀 Features
- Predicts IPL match winners
- Machine Learning model integration
- Interactive frontend UI
- Flask backend API
- Real-time prediction results

## 🛠️ Tech Stack
### Frontend
- React
- Vite
- Tailwind CSS

### Backend
- Python
- Flask

### Machine Learning
- Scikit-learn
- Pandas
- NumPy
